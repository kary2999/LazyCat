# LazyCat 使用说明 🐱

桌面萌宠 + ToDo + Telegram 集成 + 专注遮罩 四合一 macOS 助手。打开 `LazyCat.app` 后,菜单栏右上角会出现一只小猫图标,这是所有功能的入口。

---

## 一、主窗口

点菜单栏小猫 → **打开主窗口**(或 `⌘ 0`)。

四栏布局:

| 栏 | 内容 | 操作 |
|---|------|-----|
| 左侧栏 | 视图 / 人 / 优先级 | 点击切换筛选 |
| 中列表 | 当前筛选下的任务 | 单击查看详情,双击进编辑器 |
| 右详情 | 选中任务的内容、附件、属性 | 标记完成 / 编辑 / 删除按钮在底部 |
| TG 提示 | Telegram 消息聚合 | 见下方 §三 |

### 新建任务

- 中栏右上角橙色 **＋** 按钮(或 `⌘ N`)
- 弹出 Markdown 编辑器,可填:文本、@人、优先级 T0-T2、定时提醒、贴图(直接拖拽 / 粘贴)
- 自动保存草稿,意外退出后下次打开会问你恢复

### 任务操作

- 中栏点击单选,右侧详情面板查看
- 双击中栏任意任务 → 弹出编辑窗口
- 任务行最左 ◯ → 点一下标记完成
- 详情面板底部:**✓ 标记完成** / **📝 编辑** / **🗑 删除**

---

## 二、菜单栏小猫(全部功能入口)

点小猫图标弹出菜单,分组如下:

### 主操作
- **📋 打开主窗口**
- **＋ 新建事件** — 自动聚焦输入框

### 桌面陪伴
- **🐾 桌面悬浮窗** — 切换显示 / 隐藏
- **🎭 悬浮形态** — 猫 / 橘猫 / 企鹅 / 自定义图片
- **📐 悬浮尺寸** — 10% ~ 100% 共 10 档
- **💧 悬浮透明度** — 默认 100%
- **🌀 打字抖动** — 打字时悬浮窗轻微反馈
- **🖼 重置自定义悬浮图** — 一键清掉自己导入的图,回到默认猫

### 专注模式
- **🔒 专注遮罩(全屏)** — 全屏遮挡;再按一次解开
- **🌫 遮罩透明度** — 0% (完全透明,只挡毛玻璃)~ 100%(全黑)
- **🔠 遮罩字号** — 小 / 中 / 大 / 超大
- **⌨ 锁屏快捷键** — 默认 ⌥1,可改其它组合或关掉
- **💤 自动锁屏(闲置)** — 选 10 秒 / 30 秒(默认)/ 1 分 / 2 分 / 5 分 / 10 分;键盘鼠标都没动到这个时长就自动展开遮罩
- **🛡 申请辅助功能权限** — 一次性授权,授完后遮罩才能系统级吞键盘鼠标(否则只能吞本 app 内的输入)

### 外观与提醒
- **🎨 背景透明度** — 主窗口背景透明度
- **🔔 未完成汇总提醒** — 每 30 分钟弹一次未完成任务数(可关)

### 打字统计
- **📊 今日打字 X 次 · 30 日图表** — 弹出柱状图
- **🔑 申请打字监控权限** — 第一次必须给

### 其它
- **关于 LazyCat** — 版本号
- **📁 在 Finder 中显示数据**
- **🚪 退出 LazyCat** — `⌘ Q`

## 三、TG 提示(主窗口最右栏)

### 首次配置

1. 去 `my.telegram.org` 登录,进 **API development tools**,记下 `api_id`(7-8 位数字)和 `api_hash`(32 位字符串)
2. 主窗口最右栏底部点 **⚙ 设置**
3. 填进去 → **保存凭据**
4. 输入手机号(带国家码,如 `+8613812345678`)→ **发送验证码**
5. 验证码会发到你的 Telegram 官方账号(如果你已有 TG 客户端)或短信
6. 填验证码;若开了 2FA 再填密码
7. 状态变 🟢 **已连接 · 监听中** 即成功

### 日常使用

- **4 选项卡** —— 全部 / 私聊 / 群@ / 报警(关键词:告警 / alert / 异常 / 崩溃 / 严重 / fail / error 等自动归到这里)
- **翻页** —— 每页 6 条,底部 `‹ ›` + `第 X / Y 页`
- **每张卡** —— 头像 + 名字 + 来源 pill + 文本(最多 3 行) + 🕐 时间 + ＋转任务 / 已处理
- 卡片左边 **蓝色细边 = 未读**,**红色细边 = 报警**
- 点卡片右上角 **×** = 隐藏这条
- **＋ 转任务** = 直接转成主窗口的 ToDo
  - 同一联系人 24 小时内的私聊会自动追加到同一任务里(避免一人发 10 条建 10 个 task)
  - 群消息每条独立建任务,标题用群名 + @
  - 带图的消息会把图自动当附件挂上去

### 批量处理

1. 顶部点 **选择**(变蓝高亮 → 进入批量模式)
2. 每张卡左边出现 ✓ checkbox,点击多选
3. 底部出现工具栏:**已选 N · 全选 · ✓已读 · ＋转任务 · 🗑删除**
4. 操作完点 **esc** 退出批量模式

### 头部按钮

- **全部已读** — 一键把所有未读标成已读(不删消息)
- **选择** — 进入批量模式
- 不想用 TG 集成 → 把 4 选项卡全部清空 + 在 ⚙ 设置里 **登出** 即可

## 四、桌面悬浮猫

- 启动时自动出现在桌面右上角
- **拖拽** = 移动位置(下次启动记忆)
- **单击** = 展开任务列表迷你窗
- **右上红圈数字** = 当前未完成任务数
- 收到提醒时会**轻微抖一下**

切形态 / 尺寸 / 透明度 都在菜单栏小猫 → **桌面陪伴** 下。

### 自定义图片

菜单栏 → **悬浮形态** → **选择自定义图片…**:
- 选一张本地图(PNG / JPG / HEIC 都行)
- 自动抠白底(背景是浅色或棋盘格透明会被识别清掉)
- 立即生效

不喜欢效果 → 菜单 → **重置自定义悬浮图**(回到默认猫)

## 五、专注遮罩

### 怎么开

任选一种:
- 菜单 → **专注遮罩(全屏)**
- 快捷键 **⌥1**(默认)
- **闲置自动开启** — 菜单 → 自动锁屏(闲置)→ 选时长

### 怎么关

按同一个快捷键(默认 **⌥1**)即可解开。

> 如果连按 3 次乱键(非解锁键),屏幕会全屏闪 **"Look My 👀"** 警告 —— 提醒你别走神。

### 设置

- **遮罩透明度** 0~100%:0% = 几乎全透明,只剩毛玻璃质感;100% = 全黑
- **遮罩字号** 小/中/大/超大:中央时钟和当前任务名的大小
- **锁屏快捷键**:默认 ⌥1,可改 ⌥⌘1 / ⌃⌥⌘1 等;选"关闭"= 没有解锁键(慎用,只能强制退出)

### 系统级硬封锁

需要 **辅助功能权限**(菜单 → 🛡 申请辅助功能权限),授权后遮罩开启时:
- 键盘所有按键被吞(除解锁键)
- 鼠标 / 触摸板被吞(包括滚轮、三指切 Space)
- ⚠️ Space 切换由 macOS 系统级处理,本 app 拦不掉,所以滑切到新桌面会有短暂(< 1 秒)穿透,然后遮罩跟过去

没授权也能用,只是只能吞本 app 内的事件,效力打折。

## 六、提醒

### 单任务提醒
新建/编辑任务时,**提醒** 字段选时间。到点会:
- 弹 macOS 原生通知
- 悬浮猫抖一下

### 未完成汇总提醒
菜单 → 未完成汇总提醒 → 选 每 5 / 15 / 30 / 60 分钟,或关闭。
到点弹一次"还有 N 条未完成"提示。

---

## 七、打字键数统计

- 第一次启动需要 **菜单 → 申请打字监控权限**(系统设置 → 隐私 → 输入监控 勾上 LazyCat)
- 授完后侧栏底部出现:**🔥 今日 X 键 · X% · 连续 X 天**
- **双击底部条** 或 菜单 → **今日打字 X 次 · 30 日图表** = 弹打字柱状图窗口

> 权限失效(常见于重新装 app)→ 菜单 → **🛠 权限失效?重置并重新申请**,跟引导走一遍即可恢复。

| 键 | 动作 |

## 数据在哪里

按 菜单 → **📁 在 Finder 中显示数据** 直接打开:`~/Library/Application Support/MyTodoApp/`

里面:
- `data.json` — 任务 + 历史人名 + 设置
- `images/` — 任务附件图
- `tdlib/` — Telegram 缓存(登出时可删)
- `custom.png` — 自定义悬浮图
- 日志在 `~/Library/Logs/MyTodoApp.log`

完整搬迁:复制整个 `MyTodoApp/` 目录到新 Mac 同位置即可。
