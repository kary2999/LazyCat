v3 vs 现版 改动要点（待你定版）

  • Header：「批量处理」按钮有明确边框（不再是纯文字，容易找到）；激活时变 TG 蓝底白字
  • Tab 徽章：与 Tab 文字同行内联（用 flexbox），不会浮移到第二行
  • 卡片 pill：报警 + 群 @ 双 pill 并排，中间小间隔符；私聊 pill 直接显示条数（私聊 ×3）
  • 批量栏：固定 40px 高，左侧「N已选 / 总数 全选」，右侧「已读 转任务 删除 | ✓完成」；按钮有明确背景+边框，绝对可点
  • 卡片 checkbox：仅在批量模式显示（display:none → flex），不占正常模式空间；选中后群名行左缩进跟着调整
  • 「追加到已有任务」：TG 浅蓝底色按钮（区分新建 vs 追加），私聊同人 24h 内命中时出现
  • 时间行：独立一行，🕐 前缀，在正文下方、操作按钮上方
  • 翻页栏：样式不变（已 OK），仅确认高度 32px 合适
