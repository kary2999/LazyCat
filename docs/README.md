# LazyCat 使用说明

## 安装
双击 `.dmg` → 拖到 **Applications**。首次启动右键 LazyCat 选「打开」绕过 Gatekeeper 提示。

## 用法
- **记任务**：主窗口顶部填人名（必填）→ 选优先级 `T0/T1/T2` → 写正文 → 回车
- **桌面小猫**：菜单栏猫图标 → **悬浮形态** 可切 🐱 走动猫 / 🐈 橘猫 / 🐧 企鹅 / 🖼️ 自定义图
- **一键锁屏**：按 `⌥1` → 所有屏幕变磨砂；再按一次 `⌥1` 解锁（无密码、防偷看）
- **打字统计**：菜单栏 → **申请打字监控权限** → 系统弹窗点「允许」→ 退出再开 LazyCat。每天 0 点自动归零
- **背景透明度 / 悬浮尺寸 / 打字抖动**：都在菜单栏猫图标下拉里调

## 数据
任务存在 `~/Library/Application Support/MyTodoApp/data.json`，纯 JSON 可备份/迁移。

完整说明见同目录 `LazyCat-使用说明.pdf`。
