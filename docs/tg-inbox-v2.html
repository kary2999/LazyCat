实装清单 (按你 6 条对位)

① UI 字 / 按钮调粒度

字号统一 13px 主文本 / 11px 辅文本;按钮 padding 6px 12px、圆角 6px、字 11.5px / 700;NSTextField 字渲染加 allowsExpansionToolTips=false + 系统字体 .systemFont 而不是默认。Retina 下要用 .systemFont(ofSize: 13, weight: .medium) 才不糊。

② 悬浮猫终极方案

已离线把 Assets/cat.png 裁成 476×476 居中正方版,运行时直接用 NSImageView 默认 aspect-fit,不再 trim/resize 后处理。1.10.2 起生效。

③ 同人私聊追加

转任务时:在 Store 里按 person + chatType=private + createdAt 距今<24h 找已有任务。命中 → 追加 messages[],并把 +N 角标显示;否则建新任务。任务标题以群名/用户名 + 智能截断(40 字) 自动取,可手动改。

④ 时间独立一行

从 row1 的 timeLbl 拆出来,放在 text 之后单独一行,prefix 一个 🕐 + "X 分前 / X 小时前 / 昨天 14:08"。视觉层级:名字 → 来源 pill → 文本 → 时间 → 动作按钮。

⑤ 三选项卡(全部 / 私聊 / 群@ / 报警)

NSSegmentedControl 顶部插一行;过滤逻辑用 enum TGTab { all, dm, group, alert }。报警匹配正则:/告警|alert|alarm|报警|异常|exception|fail|崩溃|严重|critical/i。tab 上显示对应数量徽章。

⑥ 列表密度

卡片自适应高度(110-180px,带图更高),卡间距 8px,边框圆角 10px。880px 高的栏目能塞 6-7 张卡;支持滚动,不限上限。未读卡左边 3px 蓝边,报警 3px 红边。

⑦ 翻页

每页 6 条,底部页码栏:‹ 1 2 ... 9 ›。当前页高亮 TG 蓝;dots 表示中间省略;右侧显示"第 X / Y 页 · 共 N 条"。切 tab 时自动回到第 1 页;新消息到达时:如果在第 1 页则自动插入,否则角标 +1 但不打断当前阅读位置。

⑧ 批量处理

顶部「✓ 选择」按钮切批量模式。开启后每张卡左侧出现 checkbox,选中状态用 TG 浅蓝底色标识。底部出现 TG 蓝色工具栏:已选 N · 全选 / 批量已读 / 批量转任务 / 批量删除 / 取消。批量转任务遵循"同人私聊追加"逻辑(多人多类型分别开任务,同人私聊聚合到一个)。退出批量模式自动清空选择。
