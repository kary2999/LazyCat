🐱 LazyCat 使用说明

LazyCat 是一只长在你 Mac 桌面上的小猫，它替你记任务、提醒你、统计你今天敲了多少键。功能很多但都很简单，下面挨个讲。

给谁看：第一次拿到 LazyCat 的人。这份是「不写一行代码也能看懂」的版本。

1. 怎么把 LazyCat 装进电脑

拿到 LazyCat-x.x.x.dmg 后：

  1. 双击 .dmg，会出现一个窗口
  2. 把里面那只 LazyCat 拖到旁边的 Applications 文件夹
  3. 到「应用程序」里双击 LazyCat 启动
第一次启动会被 macOS 拦下来 ✗：因为 LazyCat 没花钱买 Apple 开发者证书，macOS 会说「无法验证开发者」。

正确做法：右键点 LazyCat → 选「打开」→ 弹出对话框上再点「打开」。这一次以后再启动就不弹了。

2. LazyCat 都能干啥

功能怎么用
📝 记任务主窗口顶部输入「@张三」+ 选优先级 + 写正文 → 回车
🔔 提醒每天每小时（可配）弹一次「你还有 N 件没做」
🐱 桌面小猫右上角有只动的小猫陪你；可换成橘猫、企鹅、或你自己的图
🔒 一键锁屏按 ⌥1 → 所有屏幕变黑磨砂；再按一次解锁
⌨️ 打字统计统计你今天一共按了多少次键，菜单栏小猫旁边实时显示
🌙 始终深色不论 Mac 系统是浅色还是深色，LazyCat 永远深色

3. 主窗口怎么用

记一条任务

  1. 主窗口顶部第一行：写人名（必填）
  2. 选优先级：无 / T2 / T1 / T0（T0 最重要）
  3. 下方大格写正文（最多 2000 字，可贴图）
  4. 按 回车 或点「添加」

列表上的标识

  • 右上角 红 / 橙 / 灰 / 绿色块：分别是滞留 7 天 / 3-6 天 / 1-2 天 / 已完成耗时
  • 双击圆圈完成任务（单击只是确认）
  • 点优先级胶囊：可改优先级 / 改人名 / 设提醒
  • 点任务正文：打开详情大窗（可继续写、贴图、改备注）
  • 点橘色 「@未指定」：补填人名（旧任务忘填可这样补）

4. 桌面小猫怎么调教

菜单栏 顶部右侧有只小猫图标，点它出菜单：

菜单项作用
桌面悬浮窗开 / 关那只小猫
悬浮形态选 🐱 走动小猫 / 🐈 橘色小猫 / 🐧 招手企鹅 / 🖼️ 自定义图片
选择自定义图片…挑你电脑上任意一张图当桌面伙伴（自动抠白底）
悬浮尺寸10 % - 100 %，10 档
悬浮透明度30 % - 100 %（小红点不会变透明）
打字抖动关闭 / 上下轻 / 上下重 / 左右轻 / 左右重

5. 一键锁屏（防别人偷看）

默认快捷键 ⌥1（Option + 1）

  • 任何 app 在前台都能按；按下后所有屏幕变成磨砂遮罩，时间和当前任务居中显示
  • 再按一次 ⌥1 → 解锁
  • 遮罩里不显示快捷键提示（防止陌生人看到一眼就解开）
  • 不喜欢 ⌥1？菜单 → 锁屏快捷键，可换 ⌥2、⇧⌘L、F13 等十几个预设
注意：菜单里如果选了「关闭（不绑定）」，就再没办法解锁，只能强退 LazyCat。

6. 今日打字数 ⌨️

LazyCat 会偷偷记你今天按了多少键（只数次数，不记内容）。

  • 菜单栏小猫图标右边会出现数字，每按一下就涨
  • 每天 0 点自动归零，旧数据保留
  • 快速打字时，桌面小猫会暂时变成大数字 → 停下 1.5 秒后变回小猫

第一次必须给「输入监控」权限

步骤：
  1. 菜单栏小猫 → 「申请打字监控权限…」
  2. 系统弹「允许 LazyCat 监听键盘？」→ 点 允许
  3. 菜单栏小猫 → 退出 LazyCat
  4. 重新打开 LazyCat（Dock 或 Applications）
  5. 敲两个字试试，数字应该开始涨

权限给了但不生效（每次升级会遇到）

这是 ad-hoc 签名 app 的通病。每次升级 LazyCat 的内部「指纹」都会变，macOS 的旧授权会失效。修法：
  1. 菜单栏小猫 → 「权限失效？重置并重新申请…」
  2. 弹出诊断窗 → 点 「打开输入监控面板」
  3. 在「输入监控」列表里找到 LazyCat 那一条 → 点旁边的【—】把整条删掉（关键！不是取消勾选，是删除）
  4. 关闭系统设置
  5. 菜单栏小猫 → 退出 LazyCat → 重新打开
  6. 系统会重新弹「允许 LazyCat 监听键盘？」→ 点 允许
  7. 再退出 LazyCat 一次 → 重新打开
  8. 数字应该开始涨了

7. 主窗口的小贴士

  • 菜单栏小猫 → 背景透明度可调 5%-100%；调到很低连任务卡片都半透，看着像贴在桌面上
  • 菜单栏小猫 → 关于 LazyCat 可以看到：版本号 / 构建时间 / app 路径，每次更新后看一眼能确认装没装上
  • 菜单栏小猫 → 在 Finder 中显示数据：你的所有任务存在 ~/Library/Application Support/MyTodoApp/data.json，纯 JSON 想备份直接拷

8. 想换头像？

菜单栏小猫 → 悬浮形态 → 选择自定义图片…

  • 选你电脑上任何一张图（PNG/JPG/HEIC 都行）
  • LazyCat 会自动把白底抠掉变透明
  • 悬浮形态自动切到「🖼️ 自定义图片」

或者高级玩法：把图改名为

  • cat-orange.png → 替换橘猫
  • penguin.png → 替换企鹅
  • cat-orange-01.png ~ cat-orange-24.png（24 张）→ 24 帧动画

放进菜单栏小猫 → 「打开自定义形象图片目录…」那个文件夹里。

9. 常见问题

启动后看到的是「事件记录」/「滴答清单 Lite」而不是 LazyCat

说明你装的是老版本，去最新 dmg 重装。

多屏没全锁住

v1.7+ 已经修，每个屏都会盖上遮罩。如果还有问题：

  • 检查系统设置 → 桌面与 Dock → 「显示器各自有独立空间」要打开
  • 把 LazyCat 退出再开一次

升级后悬浮窗位置乱了 / 图片偏离

v1.7.10+ 已经修。如果还偏：菜单栏小猫 → 桌面悬浮窗（关再开）。

悬浮窗一直挡视线

菜单栏小猫 → 悬浮透明度 → 30%（半透）；或者把悬浮尺寸调到 30% / 20%。

不想用打字统计

那就不给「输入监控」权限。LazyCat 其它功能（任务、悬浮、锁屏）全部正常工作，只是「今日打字 N 次」永远显示 0。

10. 数据存在哪里

东西路径
任务数据~/Library/Application Support/MyTodoApp/data.json
任务里的图片~/Library/Application Support/MyTodoApp/images/
自定义形象图片~/Library/Application Support/MyTodoApp/{cat,cat-orange,penguin,custom}.png
设置（透明度、尺寸、形态等）UserDefaults，com.local.mytodo domain
打字数UserDefaults，按日期分 key（如 MyTodo.typingCount.2026-04-28）
运行日志（出问题给开发看）/tmp/mytodo.log

11. 升级

新版 dmg 拖到 Applications 直接覆盖即可。任务数据不会丢（因为存在用户目录，不在 .app 里）。

升级后「输入监控」权限会失效，按 §6 末尾的步骤重新申请。这是 macOS 安全机制，不是 LazyCat 的 bug。
