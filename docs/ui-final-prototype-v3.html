展示要点

  • 列表是 固定高度 460pt 的滚动容器（高度由窗口决定，不是无限延伸）
  • 分组标题 sticky 吸顶：滚动时「⏰ 今天」「📅 明天」始终固定在列表顶部
  • 自定义 橙色细滚动条（hover 变深橙），不再用系统灰色
  • 列表底部"↓ 还有 5 条" 浮动提示，滚到底自动消失
  • 长任务文字默认单行 ellipsis；点击行 → 原地展开 4 行 + 操作按钮（不打开新窗口）
  • "散列" — 每行不强行对齐网格，按内容自然显示（meta / icons / when 紧贴右边）
  • tab 旁加搜索框 (🔍)，长列表时不用滚找
